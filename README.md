# TinyURL
Fully scalable and distibuted URL shortner using distibuted counter.</br></br>
### Technology Used
1. Spring Boot
2. MongoDB

### URL Mapping <br/>
Use /app/mapUrl end point to map your URLs which returns a token, for example:  <br/>
